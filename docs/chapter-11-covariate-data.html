### Linear

```{r}
jump_weight_tbl <- tibble(x = abs(rnorm(21, 3, 4)),
                          y = 10 + 1.2 * x + rnorm(21, 0, 2)) |>
  mutate_all(round, 1) |> 
  rename(weight = x, jumplength = y)
```

jump_weight_tbl |> 
  ggplot(aes(x = weight, y = jumplength, fill = col_pal(1, 0.5))) +
